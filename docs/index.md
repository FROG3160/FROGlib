# Welcome to FROGlib

FROGlib is a hardware abstraction and utility library created for FIRST Robotics Competition (FRC) Team 3160, the FROGs.

It provides a higher-level, object-oriented approach to interacting with common FRC hardware components like motors, sensors, and swerve drive modules.

## Features

- **Hardware Abstraction:** Wrappers for popular FRC hardware (CTRE, REV, SDS) with type hinting.
- **Swerve Drive:** Built-in classes for calculating swerve kinematics and managing swerve modules.
- **Sensors:** Abstractions for IMUs (NavX) and vision systems.
- **Utilities:** Helper functions and structures for robot positioning and mathematics.

Check out the [Getting Started](getting_started.md) guide to learn how to install and use FROGlib in your RobotPy project.
