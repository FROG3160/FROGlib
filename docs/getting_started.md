# Getting Started

## Installation

You can install FROGlib directly into your RobotPy environment.

If you have built the library locally, you can install it using pip:

```bash
pip install -e .
```

Or, if distributing via wheels, you can install the built `.whl` file.

## Basic Usage

Here is a quick example of importing and using components from FROGlib:

```python
from froglib.motors import FROGTalonFXConfig
from froglib.sensors import FROGGyro

# Example instantiation
# config = FROGTalonFXConfig(...)
# gyro = FROGGyro(...)
```

Explore the **API Reference** section to see detailed documentation for all available modules.
